# 薯条清单（FriesTodo）隐私政策

**最后更新：2026-05-15**

感谢你使用「薯条清单」。我们非常重视你的隐私。本政策说明本应用对个人数据的处理方式。简而言之：**我们不收集你的任何数据。**

## 一、我们不收集的内容

「薯条清单」**不会**收集、上传、存储或分享以下任何信息：

- 个人身份信息（姓名、邮箱、手机号、账号等）
- 设备标识符（IDFA、IDFV、广告标识符等）
- 位置、相机、麦克风、通讯录、相册、健康等任何系统权限数据
- 应用使用行为、点击事件、崩溃日志或性能指标
- 你在应用内创建的任务文本、完成时间或任何其他内容

## 二、数据如何存储

你在「薯条清单」中创建的所有任务记录，仅保存在你本机的应用沙盒中，使用系统提供的本地存储（SwiftData / SQLite）。这些数据：

- **完全离线**：应用不会发起任何网络请求，无需联网即可正常使用
- **不会上传**：开发者及任何第三方均无法访问你设备上的任务内容
- **随应用卸载消失**：删除应用即清除所有本地数据

## 三、不使用第三方服务

「薯条清单」**不集成任何**第三方 SDK，包括但不限于：

- 分析与统计（Firebase、友盟、神策、Mixpanel 等）
- 广告与归因（AdMob、AppsFlyer、Adjust 等）
- 推送与远程配置
- 崩溃报告（Crashlytics、Sentry 等）
- 社交登录或分享

## 四、不追踪、不画像

本应用不进行跨应用或跨网站的用户追踪，因此不会请求 App Tracking Transparency（ATT）授权，也不会向 Apple 报告任何追踪域名。

## 五、儿童隐私

本应用不专门面向 13 岁以下儿童，也不会有意收集儿童的个人信息。由于本应用本身就不收集任何数据，使用本应用对所有年龄段用户都不存在隐私风险。

## 六、政策变更

如未来政策有任何调整，新版本会更新本页面顶部的"最后更新"日期，并通过新版本的应用说明告知你。重大变更不会追溯应用至旧版本数据（因为旧版本也未曾收集任何数据）。

**English Summary**

FriesTodo is a fully local to-do app. It collects no personal data, makes no network requests, integrates no third-party SDKs, requests no tracking permission, and stores all task data only in the on-device application sandbox. Uninstalling the app removes all data.
