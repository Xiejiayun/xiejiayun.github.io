## "Linux用户的MacBook Pro"

Framework刚刚发布了Laptop 13 Pro，官方定位是"Linux用户的MacBook Pro"。搭载Intel Core Ultra 3处理器、LPCAMM2可更换内存、以及Framework标志性的全模块化设计。

Wired的评价是"a better, more take-apartable laptop"。Lobsters社区热议其LPCAMM2内存支持。The Verge甚至为它做了专门的发布会直播。

但这篇文章不是一个硬件评测。**我想探讨的是：在AI重塑计算范式的2026年，Framework代表的模块化硬件理念为什么比以往任何时候都更重要。**

## 被焊死的时代

过去十年，笔记本电脑行业的趋势是**焊死一切**：

- **内存焊死**：苹果从2012年开始在MacBook上焊接内存
- **存储焊死**：越来越多的超薄本将SSD直接焊在主板上
- **电池粘死**：强力胶让电池更换成为专业维修才能完成的任务
- **端口消失**：USB-A、HDMI、SD卡槽一个个被"勇气"消灭

厂商的理由是：更薄、更轻、更好的结构强度。但真正的原因是：**焊死的硬件创造了更快的换机周期和更高的维修收入**。

消费电子行业的脏秘密是：你买的不是一台电脑，而是一个有预设寿命的消费品。当你的16GB内存在2026年不够用时，你别无选择，只能买一台新的。

## Framework的反叛

Framework Laptop 13 Pro的核心设计原则：

| 组件 | 传统笔记本 | Framework 13 Pro |
|------|-----------|-----------------|
| 内存 | 焊接、不可升级 | LPCAMM2插槽、可更换 |
| 存储 | 焊接或单一M.2 | 标准M.2插槽 |
| 端口 | 固定配置 | 模块化扩展卡系统 |
| 电池 | 粘合、难更换 | 螺丝固定、5分钟更换 |
| 屏幕 | 不可更换 | 磁吸连接、可升级 |
| 主板 | 整机淘汰 | 可单独升级 |

**LPCAMM2的引入是这一代最重要的升级。**LPCAMM2是一种新的内存标准，它在保持焊接内存的性能和功耗优势的同时，实现了模块化可更换。这从技术上打破了"高性能必须焊死"的行业迷思。

Framework还同步发布了其首款eGPU产品线，通过专用接口将笔记本变成桌面级工作站。这意味着你可以用一台Framework笔记本覆盖移动办公、桌面开发、甚至轻量AI推理的全场景需求。

## AI时代为什么更需要模块化

### 内存焦虑

AI工作负载对内存的需求正在爆炸式增长。本地运行一个7B参数的大模型需要至少16GB内存，13B模型需要32GB，而更大的模型需要64GB甚至更多。

NVIDIA最近展示的本地AI推理方案（通过Spark框架加速Gemma 4），对内存的需求远超传统应用。Sebastian Raschka在"Components of A Coding Agent"中详细分析了AI编码Agent的资源消耗——仅代码上下文索引一项就可能占用数GB内存。

今天买了16GB内存的焊死笔记本，明年就可能因为无法运行本地AI而被淘汰。而一台Framework笔记本，只需要换一条内存就能续命3年。

### NPU的升级路径

AI加速硬件（NPU/Neural Engine）正在快速迭代。Intel的Core Ultra 3集成了新一代NPU，但明年的Core Ultra 4、后年的Core Ultra 5必然会有数倍的性能提升。

传统笔记本的用户将被迫每2-3年换一台新机器来获得最新的NPU。而Framework的模块化主板设计允许**只更换主板**，保留屏幕、键盘、电池、机壳等完好组件。这不仅省钱，更重要的是**减少电子垃圾**。

### 全球电子垃圾的规模已经触目惊心

- 每年全球产生超过6200万吨电子垃圾
- 笔记本电脑的平均使用寿命仅3-5年
- 其中超过80%的组件在设备报废时仍然功能完好

Framework的模块化设计直接回应了这个问题：**为什么要因为一个组件的过时而丢弃整台电脑？**

## 模块化硬件的商业模式挑战

坦率地说，Framework面临巨大的商业挑战。

**问题一：规模经济的劣势**

模块化设计天然比一体化设计需要更多的连接器、接口、标准化组件。这增加了物料成本和制造复杂度。Framework无法像苹果或联想那样通过数百万台的出货量来摊薄成本。

**问题二：用户基数的天花板**

愿意自己动手拆装电脑的用户，在全球PC用户中占比可能不到5%。Framework需要一个比"极客爱好者"更大的市场。

**问题三：升级零件的利润结构**

Framework的商业模式依赖于持续的模块和升级套件销售。但每卖出一个升级模块，就意味着少卖一台新电脑。这与传统PC厂商的利益结构完全相反。

## 但势头正在转变

几个值得注意的信号：

1. **欧盟Right to Repair法规**正在扩展到笔记本电脑品类。到2027年，在欧盟销售的笔记本可能被强制要求满足特定的可修复性标准。这对Framework是巨大利好。

2. **企业IT采购**开始关注设备全生命周期成本。当一台Framework可以通过升级使用7-8年，而传统笔记本只能用3-4年时，TCO（总拥有成本）的优势就显现出来了。

3. **AI本地化需求**正在创造一个Framework天然适配的场景：企业需要能持续升级内存和算力的工作站级笔记本，而不是每隔两年批量采购新设备。

4. **Framework的eGPU发布**意味着它开始涉足桌面级计算。如果成功，Framework将从一个笔记本品牌进化为一个**模块化计算生态系统**。

## 我的预判

1. **Framework将在2027年前推出服务器/工作站产品**，瞄准AI推理和开发场景。模块化设计在服务器领域的价值远大于消费级笔记本。

2. **LPCAMM2将成为行业标准**。Framework和Intel推动的这个标准，会逐步被戴尔、联想等主流厂商采纳。到2028年，内存焊死的笔记本将成为少数派。

3. **苹果不会跟进模块化**。苹果的商业模式依赖于高ASP（平均售价）和快速的换机周期，模块化与此根本矛盾。这将创造一个持久的市场分化。

4. **模块化AI工作站将成为独立开发者的标配**。随着本地AI推理需求的爆发，一台能持续升级内存和NPU的模块化电脑，将成为AI开发者的理性选择。

## 结论：反叛者的时代

Framework Laptop 13 Pro不是一台完美的笔记本。它比MacBook Pro更厚、更重、做工也没那么精致。

但它代表了一种在2026年越来越稀缺的价值观：**你应该真正拥有你的设备，而不是租用一个有预设寿命的消费品。** 在AI快速迭代硬件需求的时代，这种价值观不再是理想主义，而是实用主义。

如果你是一个经常需要跑本地AI模型的开发者，Framework可能是你目前能买到的最理性的选择。不是因为它最好，而是因为它最持久。
