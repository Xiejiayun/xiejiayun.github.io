# 术语表 · Glossary

> 32 条中英对照术语表，按文章出现顺序整理。配合原文与本篇导读阅读。

| 英文术语 | 中文译法 | 简注 |
|---|---|---|
| Posture | 姿势 | Addy 选用的关键词，区别于 mindset/workflow；指打开 prompt 那一刻的肌肉记忆 |
| Cognitive Offloading | 认知代办 | 把 *how* 委托给工具，自己保留 *what* 与最终判断 |
| Cognitive Surrender | 认知接管 / 认知投降 | Wharton 论文术语；AI 的输出直接成为人的输出，独立答案根本没有形成 |
| Cognitive Debt | 认知债 | MIT Your Brain on ChatGPT 论文术语；短期省力，长期偿付 |
| Comprehension Debt | 理解债 | Addy 自己的概念；代码量与真正被理解的代码量之间的鸿沟 |
| Randomized Controlled Trial (RCT) | 随机对照试验 | Anthropic 2026-01 论文采用的方法学 |
| Cohen's d | 科恩 d 值 | 效应量度量；本文中 d=0.738 属于中到大效应 |
| Trio | Trio 库 | Python 异步并发库，Anthropic 实验里被试要学习的目标 |
| EEG (Electroencephalography) | 脑电图 | MIT 实验里测量脑区连接强度的工具 |
| Alpha / Beta Band | Alpha / Beta 频段 | EEG 频段，与认知活跃度相关 |
| Brain-only Group | 纯人脑组 | MIT 实验三组之一，不允许使用任何工具 |
| LLM-to-Brain | LLM-切-人脑 | MIT 第四 session 的设计，把 LLM 用户突然切回纯人脑 |
| Anchoring Effect | 锚定效应 | 心理学经典概念，CHI 2026 用它解释 LLM 顺序效应 |
| Order > Amount | 顺序大于总量 | CHI 2026 核心论点 |
| Learning Mode (Claude) | 学习模式 | Anthropic 推出的苏格拉底式提问模式 |
| Study Mode (ChatGPT) | 学习模式 | OpenAI 对标 Learning Mode 的功能 |
| Guided Learning (Gemini) | 引导学习 | Google 对标 Learning Mode 的功能 |
| Socratic Questioning | 苏格拉底式提问 | 通过反问而非直接回答来引导学习 |
| Boilerplate | 样板代码 | 重复性、低判断含量的代码 |
| Hallucination | 幻觉 | LLM 自信地输出错误内容 |
| Plausible-looking Incorrect | 似是而非的错答 | 在表面上看起来合理但实际错误的输出 |
| Junior Developer | 初级工程师 | 经验通常 < 3 年 |
| Junior PR | 初级工程师的 PR | Addy 提的比喻：审 AI 输出要像审 junior PR |
| Re-derive by Hand | 手写复现 | 在不依赖 AI 的情况下重写之前的代码 |
| Ship Metric | 出货指标 | merged PR、closed ticket、shipped feature 等可见产出 |
| Learn Metric | 学习指标 | 私人的、不在系统里自动出现的能力增长 |
| UX Gravity | UX 引力 | 产品默认配置朝特定方向倾斜的结构性力量 |
| North Star Metric | 北极星指标 | 产品团队的核心 KPI |
| Time-to-PR | PR 周期 | 从开始工作到 PR 提交的时间 |
| Skill Formation | 技能形成 | Anthropic 论文标题里的关键词；指能力从零到成熟的过程 |
| Cognitive Surrender (Shaw & Nave) | 认知接管 | Wharton 2026 论文术语；与 cognitive offloading 严格区分 |
| Friday Question | 周五问句 | 本导读提出的总结："这周我学到了什么，还是只是关了几张工单？" |
