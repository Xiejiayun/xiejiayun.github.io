# 术语对照 Glossary · The Sigmoids Won't Save You

英中对照术语表，涵盖统计学、AI 预测、流行病学与认识论。

| English | 中文 | 一句话释义 |
|---------|------|-----------|
| Sigmoid | S 形曲线 / 饱和曲线 | 形如 1/(1+e^-x) 的有界增长曲线 |
| Exponential | 指数增长 | y = a · b^x 形式，无上界 |
| Lindy effect / Lindy's Law | Lindy 律 / Lindy 效应 | 已存续越久的事物预期还能存续越久 |
| Pareto distribution | 帕累托分布 | 重尾分布族，Lindy 律的概率形式 |
| Doomsday argument | 末日论证 / 末日先验 | 基于"你不太可能恰好在历史中点观察"的反 sigmoid 论证 |
| Copernican principle | 哥白尼原理 | 自身不处于宇宙特殊位置的先验 |
| Prior (Bayesian) | 先验 / 先验分布 | 看到证据前对参数的初始分布信念 |
| Posterior | 后验 / 后验分布 | 看到证据后更新的分布 |
| Black-box prediction | 黑盒预测 | 对生成机制一无所知时的预测 |
| Explicit model | 显式模型 | 把生成机制摆上桌的预测 |
| AIC (Akaike Information Criterion) | 赤池信息准则 | 样本内拟合优度+复杂度惩罚的模型选择判据 |
| Out-of-sample prediction | 样本外预测 | 用没参与训练的数据测量预测效度 |
| Point prediction | 点预测 | 单一数值预测，不带区间 |
| Interval prediction | 区间预测 | 带置信区间或概率分布的预测 |
| Inflection point | 拐点 | 曲线二阶导数变号处 |
| METR (Model Evaluation & Threat Research) | METR 评测机构 | 维护 AI 任务时长指标的非营利组织 |
| Task time horizon | 任务时长 | AI 能独立完成的最长人类工时 |
| Time-to-double | 翻倍时间 | 指数增长的关键参数 |
| Compute scaling | 算力规模化 | 增加训练 FLOPs 的扩展路径 |
| Data wall | 数据墙 | 公开互联网文本耗尽带来的扩展极限 |
| Recursive self-improvement | 递归自我改进 | AI 改进 AI 训练的反馈循环 |
| Scaling law | 缩放定律 | 模型规模、数据量与 loss 的幂律关系 |
| Chinchilla optimal | Chinchilla 最优 | 给定算力下 N 和 D 的最优配比 |
| Sigmoid Misidentification | sigmoid 误判 | 把仍在指数增长的过程错认为已 sigmoid |
| Burden of proof | 举证责任 | 论证义务归属 |
| Frontier model | 前沿模型 | 当前能力上限附近的模型 |
| Compute-bound | 算力受限 | 性能瓶颈在 FLOPs 上 |
| Data-bound | 数据受限 | 性能瓶颈在训练数据上 |
| Paradigm shift | 范式转移 | 库恩意义上的科学/技术换代 |
| Paradigm pipeline | 范式接力链 | 多代技术 sigmoid 拼接成的整体曲线 |
